# NETCHECK

Checks basic internet connectivity using a fast ping test and returns only numeric exit codes.

---

| Code | Meaning                                      |
| ---- | -------------------------------------------- |
| 0    | Check failed.                                |
| 1    | Check passed.                                |

---

**Usage:**

```bash
./netcheck
echo $?
```
