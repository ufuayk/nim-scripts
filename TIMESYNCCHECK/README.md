# TIMESYNCCHECK

Checks if the system clock is synchronized via NTP and returns numeric exit codes.

---

| Code | Meaning                                      |
| ---- | -------------------------------------------- |
| 0    | Not synchronized                             |
| 1    | NTP synchronized                             |

---

**Usage:**

```bash
./timesynccheck
echo $?
```
